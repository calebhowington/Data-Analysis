# Data Analysis

## Resonator Analysis
To fit 'file.s2p', 
```
from ResonatorFit import plotS21, analyze
results = analyze('path/to/file.s2p')
```

The object 'results' contains the fit parameters, data, and file information. 

To only plot the file, 
```
plotS21('path/to/file.s2p')
```
